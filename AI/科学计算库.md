## virtualenv

```sh
# virtualenv
pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple virtualenv 
pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple virtualenvwrapper-win 
# 创建虚拟环境
mkvirtualenv env_name -p `which python3.6`
# 进入环境
workon env_name
# 退出环境
deactivate
```
