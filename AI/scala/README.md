### 函数式编程
Scala语言是一个完全面向对象编程语言。万物皆对象
对象的本质：对数据和行为的一个封装

Scala语言是一个完全函数式编程语言。万物皆函数
函数的本质：函数可以当做一个值进行传递

函数和方法的区别
- 未完成某一功能的程序语句的几何，称为函数
- 类中的函数称为方法
